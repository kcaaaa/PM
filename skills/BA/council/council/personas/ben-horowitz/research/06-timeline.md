# Ben Horowitz - 人物时间线

| 时间 | 事件 | 影响 |
|------|------|------|
| 1966 | 出生于伦敦，在加州伯克利长大 | 在嬉皮士社区和多元文化环境中长大，奠定开放心态 |
| ~1970s | 父亲David Horowitz从左翼转右翼 | 目睹意识形态转变——学会了质疑任何"正确"的叙事 |
| ~1980s | 高中和大学时期深度接触嘻哈文化 | 嘻哈成为终身文化认同：真实性、生存、不装 |
| 1988 | Columbia University计算机科学学位 | 技术基础 |
| 1990 | UCLA计算机科学硕士 | 深化技术能力 |
| ~1990 | 加入Silicon Graphics (SGI) | 第一份科技行业工作，接触硬核工程文化 |
| 1995 | 加入Netscape | 遇到Marc Andreessen。互联网革命第一线。学会了"在高速增长中做决策" |
| 1995-98 | Netscape多个产品经理/VP角色 | 写了"Good Product Manager / Bad Product Manager"——成为硅谷PM圣经 |
| 1998 | Netscape被AOL收购 | 目睹大公司吞噬创业公司的过程。种下日后"保护创始人"的种子 |
| 1998-99 | AOL时期短暂工作 | 大公司政治和文化冲突的第一手经验 |
| 1999 | 与Marc Andreessen共同创立Loudcloud | 云计算概念先驱（比AWS早7年）。开始CEO生涯 |
| 2000 | 互联网泡沫破裂 | 公司从高估值瞬间面临生存危机。"The Struggle"开始 |
| 2001.3 | Loudcloud在泡沫最低点IPO | 在"不可能"的时间完成IPO——"没有好选项时选不死的选项" |
| 2002 | 将托管业务卖给EDS，转型为Opsware | 股价跌至$0.35。从数百人裁到核心团队。最黑暗的时期 |
| 2002-07 | Opsware求生和增长 | 5年的战时CEO经历——"每天都在吃玻璃碎片" |
| 2003 | 差点被BladeLogic等竞争对手灭掉 | "Lead bullets, not silver bullets"——产品基本功是唯一出路 |
| 2007 | HP以16亿美元收购Opsware | 从$0.35到$14.25/股。验证了战时CEO方法论 |
| 2009 | 与Marc Andreessen共同创立a16z | 用CEO经验重新定义VC——"VC即服务"，不只给钱给运营支持 |
| 2009-12 | a16z早期投资：Skype, Facebook, Airbnb, GitHub等 | 建立投资声誉。关注创始人的"战时能力" |
| 2012 | a16z投资GitHub | 2018年被微软以75亿美元收购 |
| 2013 | a16z领投Coinbase A轮 | crypto战略布局开始 |
| 2014 | 出版《The Hard Thing About Hard Things》 | 成为全球创业者必读书。以嘻哈歌词开每一章 |
| ~2015 | a16z开始大规模crypto投资 | 成为crypto领域最大机构投资者之一，也引来争议 |
| 2019 | 出版《What You Do Is Who You Are》 | 从历史案例（海地革命、武士道等）提炼企业文化方法论 |
| 2021 | a16z AUM超过350亿美元 | 从新锐基金成为mega-fund |
| 2021 | Coinbase直接上市 | a16z的crypto投资获得巨额回报 |
| ~2022-23 | Crypto冬天 | a16z的crypto投资面临质疑，但继续加注 |
| 2023-24 | a16z政治参与增加 | 与Marc一起积极参与政策游说和政治捐款，引发讨论 |
| 2024 | a16z继续扩张，AI投资加速 | 从crypto转向AI的战略调整 |
| 2025-26 | a16z持续作为顶级VC | 继续在AI、crypto、bio等领域布局 |

## 人生阶段总结

### 第一阶段：学习期（1966-1999）
技术训练 + 嘻哈文化 + Netscape初体验 + 遇到Marc

### 第二阶段：战争期（1999-2007）
Loudcloud/Opsware的8年CEO生涯——所有后来的思想都来自这个阶段

### 第三阶段：VC/思想家期（2009-至今）
将CEO经验转化为投资框架和管理思想。著书立说。建立a16z平台

### 贯穿始终的线索
- 嘻哈文化：从少年到60岁，始终是身份认同的一部分
- 与Marc的搭档关系：从Netscape到a16z，30年战友
- "The Struggle"：人生和事业中反复出现的主题——挣扎是常态，不是例外
