# 马云 · 人物时间线

## 完整时间线

| 时间 | 事件 | 意义 |
|------|------|------|
| 1964.9.10 | 出生于浙江杭州 | 教师节出生，后来以教师身份为荣 |
| ~1972-1980 | 文化大革命期间成长 | 经历物质匮乏和社会动荡，培养韧性 |
| ~1979 | 在西湖边结识外国游客，开始自学英语 | 语言能力成为打开世界的钥匙 |
| 1982 | 第一次高考失败（数学1分） | 反复失败的原型经历 |
| 1983 | 第二次高考失败 | 继续坚持 |
| 1984 | 第三次高考，考入杭州师范学院英语专业 | "三次高考"成为其最著名的励志故事 |
| 1988 | 毕业，被分配到杭州电子工业学院任英语讲师 | 月薪约12美元，教师身份成为终身认同 |
| 1994 | 创办翻译社，同年启动网站 | 第一次创业尝试 |
| 1995.1 | 首次访美（西雅图），第一次接触互联网 | 人生转折点——在朋友办公室搜索"beer"，发现网上没有中国信息 |
| 1995 | 创办"中国黄页"（China Yellow Pages） | 中国最早的互联网企业之一，在线商业目录 |
| 1996 | 中国黄页被中国电信收购 | 第一次创业的结局——学到了与体制合作的经验 |
| 1997-1999 | 在外经贸部做电子商务项目 | 积累B2B经验和政府关系，了解中小企业需求 |
| 1999.6.28 | 与17位朋友在杭州公寓创办阿里巴巴 | 起步资金6万美元，愿景："让天下没有难做的生意" |
| 1999.10 | 获得高盛500万美元投资 | 第一轮外部融资 |
| 2000.1 | 获得软银2000万美元投资 | 孙正义"6分钟决定投资"的传奇故事 |
| 2003 | 秘密创办淘宝（C2C）| 应对eBay进入中国，"鳄鱼vs鲨鱼"战略 |
| 2003 | 创立支付宝 | 第三方担保支付，解决电商信任问题 |
| 2004 | 淘宝在中国市场份额超过eBay | 免费策略+本地化的胜利 |
| 2006 | eBay退出中国市场 | 淘宝彻底胜出 |
| 2008 | 推出天猫（原淘宝商城）| B2C模式，品牌商入驻 |
| 2009 | 创立阿里云 | 押注云计算，当时内部争议极大 |
| 2009.11.11 | 首个"双十一"购物节 | 后来成为全球最大的在线购物活动 |
| 2010-2011 | 支付宝从阿里集团拆分 | 争议性决策——监管合规 vs 大股东利益 |
| 2013 | 创办菜鸟网络 | 物流基础设施，完善生态闭环 |
| 2014.9.19 | 阿里巴巴在纽交所IPO，融资250亿美元 | 当时全球最大IPO |
| 2015 | 创办湖畔大学 | 以"研究失败"为核心的企业家学院 |
| 2016.9 | 在G20杭州峰会推动eWTP | 将全球化普惠到中小企业的倡议 |
| 2017 | 达沃斯年会主旨演讲 | 成为中国在全球舞台最有影响力的商业领袖 |
| 2018.9.10 | 宣布将于一年后卸任 | "深思熟虑、认真准备了10年的计划" |
| 2019.4 | 公开支持996工作制 | 引发广泛社会批评 |
| 2019.8 | 与Elon Musk在WAIC辩论AI | AI乐观主义 vs AI悲观主义 |
| 2019.9.10 | 正式卸任阿里巴巴董事局主席 | 张勇接棒，"制度传承的开始" |
| 2020.10.24 | 外滩金融峰会演讲 | 批评金融监管，引发连锁反应 |
| 2020.11.2 | 被四部门联合约谈 | 蚂蚁集团IPO危机开始 |
| 2020.11.3 | 蚂蚁集团370亿美元IPO暂停 | 全球最大IPO在最后一刻被叫停 |
| 2020.12-2023.1 | 从公众视野中"消失" | 极少公开露面，偶尔在海外被拍到 |
| 2021.4 | 阿里巴巴被罚182亿元（反垄断） | 中国反垄断罚款纪录 |
| 2023.1 | 放弃蚂蚁集团控制权 | 蚂蚁整改的一部分 |
| 2023.3 | 回国，在杭州云谷学校短暂露面 | 首次回归公众视野的信号 |
| 2023.7 | 蚂蚁集团被罚71.23亿元 | 监管整改阶段性结束 |
| 2025.2.17 | 出席习近平主持的民营企业座谈会 | 政治"解冻"的标志性事件 |

## 人生演化轨迹

```
英语老师（1988-1994）
  → 中国黄页创业者（1995-1996）
    → 政府电商项目（1997-1999）
      → 阿里巴巴创始人（1999-2013）
        → 全球商业领袖/达沃斯常客（2014-2019）
          → 退休的"马老师"（2019-2020）
            → 沉默的低调者（2020-2024）
              → AI时代的回归者（2025-）
```
