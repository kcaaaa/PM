# Marc Andreessen - 时间线

## 完整时间线

| 时间 | 事件 | 意义 |
|------|------|------|
| 1971年7月9日 | 出生于爱荷华州Cedar Falls | 中西部小城背景——不是硅谷本地人 |
| 1989 | 进入UIUC（伊利诺伊大学香槟分校）计算机科学系 | 在NCSA（国家超级计算应用中心）工作 |
| 1993 | 与Eric Bina共同创建Mosaic浏览器 | 第一个主流图形化Web浏览器——改变了互联网历史 |
| 1993 | 计算机科学学士毕业 | |
| 1994 | 与Jim Clark联合创立Mosaic Communications（后更名Netscape）| 22岁，搬到硅谷 |
| 1995年8月9日 | Netscape IPO——"The IPO that launched the internet era" | 首日市值$2.9B，Marc个人身价$58M，互联网商业化的起点 |
| 1995-1998 | Browser Wars：Netscape vs Microsoft IE | IE捆绑Windows策略最终赢得市场 |
| 1998 | 美国政府对微软反垄断诉讼（部分因IE捆绑） | Marc的经历推动了对平台垄断的理解 |
| 1999年3月 | AOL以$4.2B收购Netscape | Netscape作为独立公司的终结 |
| 1999年9月 | 联合创立Loudcloud | 云计算先驱（比AWS早7年） |
| 2000-2001 | 互联网泡沫破灭 | Loudcloud在泡沫中艰难生存 |
| 2001年3月 | Loudcloud IPO——在泡沫破灭中上市 | 极其艰难的IPO，但成功融资 |
| 2002 | 出售Loudcloud的托管业务给EDS，转型为Opsware | 战略转型——从服务到软件 |
| 2007年9月 | HP以$1.6B收购Opsware | 成功退出，但远非容易——中间经历了极其痛苦的过程 |
| 2008 | 加入Facebook董事会 | 成为扎克伯格最重要的顾问之一 |
| 2009年7月 | 与Ben Horowitz联合创立Andreessen Horowitz (a16z) | 颠覆VC行业的"全栈服务"模式 |
| 2011年8月 | 发表"Why Software Is Eating the World" (WSJ) | 定义了一个时代的投资论文 |
| 2012 | Facebook IPO | a16z作为早期投资者获得巨额回报 |
| 2013-2018 | a16z快速扩张，管理资产从$300M到$11B+ | 成为硅谷最大VC之一 |
| 2018 | a16z注册为Registered Investment Advisor (RIA) | 允许更大比例投资crypto等非传统资产 |
| 2020年4月 | 发表"It's Time to Build" | COVID期间的檄文，从技术乐观到制度批判 |
| 2021 | a16z crypto基金达到$2.2B | 加密货币投资大幅扩张 |
| 2021 | Coinbase IPO（a16z最大crypto退出之一）| 验证了crypto投资论文 |
| 2022 | Crypto Winter | a16z crypto投资组合大幅缩水，但坚持长期持有 |
| 2023年6月 | 发表"Why AI Will Save the World" | 对AI恐慌论的正面回应 |
| 2023年10月 | 发表"The Techno-Optimist Manifesto" | 最全面的世界观宣言——引发巨大争议 |
| 2024年7月 | 公开支持Donald Trump竞选总统 | 与Ben Horowitz一起成为科技界最知名的Trump支持者 |
| 2024年11月 | Trump赢得大选 | Marc的政治赌注获胜——预期更友好的科技监管环境 |

## 最近12个月动态（2025年4月-2026年4月）

- **a16z继续大规模AI投资**：在AI基础设施、AI应用、AI代理（agents）领域积极布局
- **政治影响力**：在Trump第二任期中，a16z与政府关系更加紧密，推动crypto友好监管和AI轻触监管
- **AI政策辩论**：Marc继续反对AI过度监管，在各种论坛和播客中为AI加速主义辩护
- **a16z规模持续扩张**：管理资产规模继续增长
- **Twitter/X活跃度**：继续在X上高频发布关于AI、政治、技术的内容
- **Crypto回暖**：随着监管环境改善和市场回暖，a16z的crypto投资组合价值回升
- **与AI安全社区的持续辩论**：在AI加速vs安全的辩论中继续站在加速一方

## 关键人生转折点

1. **Mosaic/Netscape（1993-1995）**：从中西部小子到互联网先驱——证明了技术可以改变世界
2. **Browser Wars失败（1998-1999）**：学到分发渠道的力量——技术不够，平台才是关键
3. **Loudcloud地狱（2000-2002）**：在绝境中生存——建设者精神的终极考验
4. **转型VC（2009）**：从建设者到投资者/思想领袖——杠杆从代码变成资本+叙事
5. **政治觉醒（2020-2024）**：从技术乐观主义到政治参与——意识到制度是技术进步的最大障碍
