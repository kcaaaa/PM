# Eric Ries - 人物时间线

## 完整时间线（1978年至今）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1978 | 出生于美国 | — |
| ~1996 | 进入Yale大学，主修计算机科学 | 技术背景的起点 |
| ~2000 | Yale毕业，进入硅谷 | 正值互联网泡沫破裂后 |
| 2001 | 第一次创业（Catalyst Recruiting，在线招聘） | 泡沫破裂时期创业，失败但积累经验 |
| 2001-2003 | 在There.com工作（3D虚拟世界） | 认识Will Harvey，后来的IMVU联合创始人 |
| 2004 | 与Will Harvey共同创立IMVU | 精益创业方法论的实验室 |
| 2004 | 成为Steve Blank在UC Berkeley Haas商学院的学生 | 接触Customer Development方法论，对其思维产生深远影响 |
| 2005 | 在IMVU实践持续部署（每天50+次） | 速度实验——缩短Build-Measure-Learn循环 |
| 2005-2006 | IMVU的IM互操作功能失败 | "花6个月构建没人要的东西"——精益创业的原点故事 |
| 2007 | IMVU转型成功，找到PMF（付费3D虚拟形象） | 验证了Pivot和MVP的力量 |
| 2008 | 开始写Startup Lessons Learned博客 | 从实践者转向方法论提炼者 |
| 2008 | 首次公开使用"Lean Startup"这个术语 | 命名的力量——一个好名字帮助概念传播 |
| 2008-2009 | 在Steve Blank课堂上教授精益创业 | 师生变为同行——Blank教Customer Development，Ries教Lean Startup |
| 2009 | 精益创业概念开始在硅谷获得关注 | 从博客到运动的转折点 |
| 2010 | 首届Lean Startup Conference | 精益创业从概念变为运动 |
| 2011 | 出版《The Lean Startup》 | 现象级畅销书，全球影响 |
| 2011-2014 | 全球巡回演讲、咨询、培训 | "传教士"阶段，精益创业传播到创业之外的领域 |
| 2013 | GE采用精益创业方法论（FastWorks项目） | 证明大企业也能用精益方法——但后来被质疑效果 |
| 2015 | 开始筹备LTSE（长期股票交易所） | 从方法论回归创业，但目标更大——制度变革 |
| 2017 | 出版《The Startup Way》 | 将精益创业应用于大企业，反响不如第一本 |
| 2019 | LTSE获得SEC批准 | 十年来美国首个新获批的全国性股票交易所 |
| 2020 | LTSE正式运营 | 开始接受上市申请 |
| 2020-2023 | LTSE面临增长挑战 | 上市公司数量和交易量低于预期 |
| 2023 | LTSE经营困难 | 理念与市场现实的差距 |
| 2024-2026 | 继续推动LTSE和精益创业教育 | 方法论持续在全球创业教育中占据核心位置 |

## 关键转折点

### 转折1: IMVU的IM互操作失败（~2005）
- 从"凭直觉构建"到"先验证再构建"的认知转变
- 如果没有这次失败，就没有精益创业

### 转折2: 遇见Steve Blank（2004）
- Customer Development方法论给了Ries一个理论框架来理解自己的实践
- 师生关系定义了Ries的知识谱系

### 转折3: 《The Lean Startup》出版（2011）
- 从小众博客到全球现象
- Ries的身份从"创业者/工程师"永久性地转变为"方法论传教士"

### 转折4: LTSE创立（2015）
- 从"教别人怎么做"回归"自己做"
- 将精益创业的视野从产品层面扩展到制度层面
- 也是一次自我检验——能否用自己的方法论创建一个成功的企业？

## 人物成长弧线

```
技术创业者（2000-2007）
    → 方法论提炼者（2008-2011）
        → 运动领袖/传教士（2011-2015）
            → 制度创业者（2015-至今）
```

核心主线：减少人类创造力的浪费——从产品层面（别做没人要的东西）到制度层面（别让短期主义扼杀创新）。
