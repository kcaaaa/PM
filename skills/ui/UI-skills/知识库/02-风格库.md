## Part A · 主表（67 种风格一览）

> - **L/D** = Light Mode / Dark Mode 支持（✓完全 / ◐部分 / ✗不）

### B.1 Minimalism & Swiss Style
- **CSS 关键词**：`display: grid; gap: 2rem; font-family: sans-serif; color: #000 or #FFF; max-width: 1200px; clean borders; no box-shadow unless necessary`
- **Checklist**：

### B.2 Neumorphism

### B.3 Glassmorphism

- **Checklist**：
  - ☐ backdrop-filter blur 10-20px
  - ☐ 半透明白 15-30% 不透明度
  - ☐ 细边 1px light
  - ☐ 鲜艳背景验证

### B.4 Brutalism

### B.5 Dark Mode (OLED)

## 使用提示

- 先拿到《01》的 `Style_Priority`（如 "Glassmorphism + Flat Design"），**主风格取 "+" 左侧**。
- 在主表 A 找到主风格 → 校验 **Best For / Do Not Use For** 与当前产品是否一致；不一致时改用次风格。
- 在 **Part B** 查 CSS 关键词、实现 Checklist；复合场景（例如"Glassmorphism 主 + Flat 辅"）时，以主风格的 Variables 为基线，卡片外壳用 Glassmorphism，按钮和图标用 Flat Design。
- 在 **Part C** 摘取 Primary Colors，再去《03-配色方案.md》查产品维度的 Primary/Secondary/CTA/BG/Text 具体 hex。
- 在 **Part D** 若需生图，使用对应 Prompt Keywords。
