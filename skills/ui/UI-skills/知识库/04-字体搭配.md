## 使用提示

1. 从《01》拿到 `Typography_Mood`（如 "Elegant + Refined typography"），把其中的气质词代入 Part A 的 **Mood/Style Keywords** 列做匹配。
2. 同时对照 **Best For** 列，确认此对字是否服务于你的产品场景。
3. 若国际化（中 / 日 / 韩 / 阿拉伯 / 越 / 泰 / 希伯来），请直接用 Part B 的 Noto 家族对应项。
4. 找到后，直接复制该行的 `@import` 或 `Tailwind Config` 到项目。

## Part F · 使用红线

1. **字重不要全部加载**：仅加载需要的 weights（通常 3-5 个）——多余字重会拖慢页面。
2. **Display / Script 字体**（如 Bebas Neue / Great Vibes / Abril Fatface）**仅用于 Hero / 标题 / 强调**，禁止用作正文。
3. **Mono + Mono 全单色**（如 Brutalist Raw）仅限极端风格；日常工程勿用，影响阅读。
4. **中英混排**时，CSS 的 `font-family` 需要 **CJK 字体在前**（或在 `unicode-range` 里指定范围），否则中文会 fallback 到系统字体。
5. **变量字体**（如 Lexend Mega `wght@100..900`）可显著减小请求数；优先采用。
6. **`display=swap`** 必加——避免 FOIT（不可见文字闪烁）。
