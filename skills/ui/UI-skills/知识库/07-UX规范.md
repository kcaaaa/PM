# 知识库 · 07 UX 规范（99 条反模式与建议）

> **来源**：由 `ux-guidelines.csv` 完整转译。
> **作用**：提供**分类清晰的 UX 红线与正确做法**，涵盖导航、动画、布局、触控、交互、无障碍、性能、表单、响应式、排印、反馈、内容、空间 UI、AI 交互、可持续性。
> **搜索字段**：`Category / Issue / Description / Platform`

---

## 严重性分级

- **High**：直接影响可用性 / 无障碍 / 主要流程 → **必须修复**
- **Medium**：影响体验质量，但不致命 → **强烈建议修复**
- **Low**：打磨类 / 细节 → **有余力再做**

## 高频红线（最容易忽略的 10 条，建议逐条核对）

1. **cursor-pointer** 必须加在所有可点击元素上（#29）。
2. **Focus ring** 绝不能 `outline: none` 又不给替代（#28）。
3. **`prefers-reduced-motion`** 必须被尊重（#9 / #99）。
4. **Alt text** 有意义图片必加（#38）。
5. **Label** 永不能被 placeholder 代替（#54）。
6. **Touch target** ≥ 44×44px（#22）。
7. **Loading state** 超过 300ms 必给反馈（#78）。
8. **Mobile font size** ≥ 16px（#67）。
9. **Viewport meta** 必配（#68）。
10. **Confirmation dialog** 不可逆操作前必加（#35）。
