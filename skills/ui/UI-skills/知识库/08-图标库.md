# 知识库 · 08 图标库（100 个 Lucide 速查）

> **来源**：由 `icons.csv` 完整转译。
> **作用**：按功能场景速查图标名 + 导入代码；**严禁用 Emoji 当 UI icon**。
> **主库**：[Lucide](https://lucide.dev)（推荐），备选 [Heroicons](https://heroicons.com)，品牌图标用 [Simple Icons](https://simpleicons.org)。
> **风格**：全部 Outline 风；所有图标统一 24×24 viewBox，渲染尺寸 `w-6 h-6`。

---

## 核心规则

1. **永远不要**用 🎨🚀⚙️💡 这类 Emoji 作为界面 icon——使用 SVG。
2. **图标集要统一**——不要 Lucide + Heroicons 混用（除非明确分区）。
3. **品牌 Logo**（Google/Apple/GitHub/Twitter 等）**必须**从 [Simple Icons](https://simpleicons.org) 取官方 SVG，不要凭记忆画。
4. 图标按钮（只有图标没有文字）必须加 `aria-label`：`<button aria-label="Close menu"><X /></button>`
5. 使用 React 时从 `lucide-react` 导入；使用纯 HTML 时可用 `<i data-lucide="name">` 再 `lucide.createIcons()`。

## 安装与使用

### React（Lucide）
```bash
npm install lucide-react
```

```jsx
import { Menu, Search, Settings } from 'lucide-react';

<button aria-label="Open menu">
  <Menu className="w-6 h-6" />
</button>
```

### HTML + CDN（Lucide）
```html
<script src="https://unpkg.com/lucide@latest"></script>
<i data-lucide="menu"></i>
<script>lucide.createIcons();</script>
```

### Tailwind 统一尺寸
- 导航 / 按钮内：`w-5 h-5`
- 主 icon 按钮：`w-6 h-6`
- Hero 区大图标：`w-10 h-10` 或 `w-12 h-12`
- 小标签内：`w-4 h-4`
