# 知识库 · 01 产品-风格映射（决策中枢）⭐

> **来源**：由原项目的 `products.csv`(96 类) + `ui-reasoning.csv`(100 条规则) 合并整理而成。
> **作用**：输入 **产品类型**，输出 **推荐风格 / 配色基调 / 字体基调 / 落地页模式 / 必须规避的反模式 / 决策规则**。
> **它是本技能包的"决策中枢"**：任何一次设计咨询都**必须**先查这张表。

---

## 使用方法（三步）

1. 用《工作流/01-需求分析.md》得到的 `product_type` 精确 / 子串 / 关键词匹配下表的 `UI_Category`
2. 抄下整行的 `Recommended_Pattern / Style_Priority / Color_Mood / Typography_Mood / Key_Effects / Decision_Rules / Anti_Patterns`
3. 把 `Anti_Patterns` **整段原样**放进最终的"必须规避"清单（红线），不可省略

> 若用户有明确限定词（luxury / children / ux-focused / real-time 等），按 `Decision_Rules` 做调权。
> 具体调权见《工作流/02-设计系统生成.md》第 3 节。

## Part D · 关键考虑（Key Considerations 摘要）

> 每个类目的"必做事项"简述。用于在最终输出里给用户一句话提醒。

- **SaaS**：在现代感和清晰度之间平衡；CTA 突出。
- **E-commerce**：高视觉层级 + 转化率 + 参与感。
- **Healthcare**：无障碍强制；宁静的视觉。
- **Fintech**：安全感外显；实时数据关键。
- **Beauty/Spa**：宁静氛围 + 预约系统 + 前后对比图 + 客户证言。
- **Government**：WCAG AAA；信任至上。
- **Gaming**：沉浸 + 性能。
- **Portfolio**：展示作品 + 个性。
- **Banking**：安全优先 + 账户概览 + 无障碍关键。
- **AI/Chatbot**：对话式 UI + 流式文字 + 减少外壳。
- **Luxury**：优雅至上；高质量素材；讲故事。
- **News/Media**：移动优先阅读 + 快速加载。
- **Education/E-learning**：进度追踪 + 游戏化 + 视频播放器。
- **Non-profit**：影响力故事 + 透明度 + 情感连接。
- ...（完整 100 行 Key Considerations 见原 `products.csv`，本处只收录最常见 14 项；必要时直接参考 Part A/B 已涵盖的 must_have 字段即可。）
